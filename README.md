# Order Composer

オーダーの発動順、効果時間、待機時間を組み立てて確認するための、非公式ファンツール試作版です。

## 起動

依存関係やビルドは不要です。`index.html` をブラウザで開いてください。

### スマホから開く

Macとスマホを同じWi-Fiに接続し、`スマホで開く.command` をダブルクリックします。表示されたスマホ用URLを、スマホのブラウザで開いてください。

Macがスリープしたり、起動したターミナルを閉じたりするとアクセスできなくなります。

ローカルサーバーを使う場合:

```sh
python3 -m http.server 4173
```

その後、`http://localhost:4173` を開きます。

## 現在できること

- オーダー名・効果の検索
- カテゴリ絞り込み
- 最大8件の編成
- PCのドラッグ＆ドロップ、スマホの長押しドラッグによる並べ替え
- 発動時刻、15分からの残り時間の自動計算
- `localStorage` への保存
- URLによる編成共有

## データの変更

試作データは `app.js` 冒頭の `ORDERS` にあります。実データを扱う場合は、名称、秒数、権利関係を確認したうえで差し替えてください。

本ツールはゲーム運営・権利者とは関係ありません。
